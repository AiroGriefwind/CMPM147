Experiment 4b - Monstrous Eyeballs

Description

Every tile is a monstrous eyeball, and they all stare at the audienceâ€™s cursor (mouseX,mouseY).

Technical

Clicking on Tiles: Triggers eye animations and plays one of two sound effects, chosen based on a hash function of the tile's coordinates. This interaction is not only about altering visuals but also about immersing the user in the auditory landscape of each tile. Cursor Movement: Influences the direction each eyeball looks, making the entire grid responsive to the user's movements, further enhancing the feeling of a living, breathing digital organism.
